# 🔢 React Counter App

A simple and interactive **Counter Web Application** built using **React.js**.
This app allows users to **increase**, **decrease**, and **reset** the counter value dynamically.

---

## 🚀 Features

* ➕ Increment counter with multiple values (+1 to +5)
* ➖ Decrement counter with multiple values (-1 to -5)
* 🔄 Reset button to bring counter back to **0**
* ⚡ Real-time updates using React state
* 🎨 Clean UI with centered layout and background design

---

## 🎯 Functionality

* Click **+ buttons** → increases counter
* Click **- buttons** → decreases counter
* Click **Reset** → sets counter to **0**

---

## 🧠 How It Works

* Uses **React `useState` hook** to manage counter value
* Each button updates state dynamically
* UI automatically re-renders when state changes

---

## 🖥️ UI Preview

* Large central display showing current count
* Buttons grouped for:

  * Positive increments
  * Negative decrements
* Reset button placed in the middle
